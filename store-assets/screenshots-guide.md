## Screenshot Guidelines

### Visual Requirements
- **Resolution**: 1280x800px (16:10 aspect ratio)
- **Format**: PNG with transparency support
- **Quality**: High resolution, crisp text
- **Browser**: Use Chrome with clean, minimal setup
- **Theme**: Light theme for better visibility

### Content Guidelines
- Show realistic usage scenarios
- Include sample data that looks authentic
- Ensure all text is readable
- Highlight key features and benefits
- Use consistent browser chrome across screenshots

### Technical Setup
1. Set browser zoom to 100%
2. Use a clean browser profile with no other extensions
3. Clear browser cache and cookies
4. Use sample data that demonstrates value
5. Ensure consistent lighting and contrast

### Sample Data to Use
- Tab limit: 5
- Mazes completed: 12
- Tabs blocked: 28
- Days active: 7
- Current session: 2 mazes

### Branding Elements
- Consistent color scheme: #667eea, #764ba2, #4ecdc4, #ff6b6b
- Professional but playful tone
- Clear, readable typography
- Proper spacing and alignment

Keep captions concise and benefit-focused:

- Use action-oriented language
