# Extension Icons

This directory contains the icons for the Chrome extension in various sizes.

## Icon Sizes Required:
- 16x16 - Small icon for extension toolbar
- 32x32 - Medium icon for extension management
- 48x48 - Icon for extension details
- 128x128 - Large icon for Chrome Web Store

## Design Notes:
The icon represents:
- A browser tab (rectangular shape)
- Maze elements (colorful maze pattern overlay)
- Tab management theme (multiple tab dividers)
- Playful yet professional appearance

## Colors:
- Primary: #667eea (purple-blue gradient)
- Secondary: #764ba2 (darker purple)
- Accent: #4ecdc4 (teal for maze elements)
- Alert: #ff6b6b (red for player/focus elements)

To generate actual PNG files from the SVG, use an SVG to PNG converter or image editing software.
